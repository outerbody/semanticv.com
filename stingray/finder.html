Adaptive Information Tools

Stingray is a dynamic semantic information tool. Stingray can...

  • Learn meanings for words based on how they are used.
  • Explore information using concepts.
  • Discover themes in a body of text.

Stingray Finder

Stingray Finder uses SemanticV's core technology for exploring large data sets using concepts. Finder is a standalone corpus analyzer and concept based search tool.

Stingray Finder enables you to analyze data within your network, developing custom semantic spaces, and explore your own data using the concept aware Stingray technology.
